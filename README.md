## このプロジェクトについて

2026 年、AI は「使えるかどうか」ではなく、**「どこまで使いこなしているか」** で人と組織の差が決まる時代になりました。
カジュアルな ChatGPT ユーザーと、自律型エージェント群を動かすエンジニアの差は、いまや **10 倍以上** にまで広がっています。

THE AI RANK は、この **"AI 熟達度"** を誰でも共通言語で語れるように設計した、**9 段階の格付けシステム** です。
10 問の診断を受けると、あなたの現在地が刻まれた **パーソナライズド認定証** が発行されます。

- **発案・監修**：[CHAEN](https://x.com/masahirochaen)（個人プロジェクト）
- **リリース**：2026 年 4 月
- **ライセンス**：MIT（格付け思想・コード共に）
- **ゴール**：このリポジトリで議論を集め、**「最強の世界共通 AI 格付けランク」** へ育てる

> 新卒・ビジネス職：**Lv.III 以上** / ナレッジワーカー：**Lv.IV 以上** / AI 関連職：**Lv.VI 以上** / AI エンジニア・起業家：**Lv.VII** / AI プロダクトで収益化：**Lv.VIII** / 未来の到達地点：**Lv.IX**

## 各レベルの詳細

### Lv.I ビギナー — 「AI に触れた者」

**定義**：汎用 AI の**無料版**を、検索・備忘録の延長として使う段階。

**具体的なタスク**
- ChatGPT 無料版で「東京で雨の日でも行けるデート先」を聞く
- Gemini に英語メールの和訳を頼む
- Copilot で文章のトーン調整を依頼
- 会議のメモを箇条書きで整えてもらう

**成果物**：5〜10 行の要約メモ／翻訳された文章／丁寧なメール文面／買い物リスト

**Before / After**
- Before：Google 検索で 15 分かけて情報収集
- After：ChatGPT に 3 回質問して 5 分で終わる

**卒業条件**：月額課金に踏み切る／1 日複数回 AI を開く習慣になる。

---

### Lv.II パワーユーザー — 「AI を使い分ける者」

**定義**：汎用 AI の**有料プラン**に加え、**用途別の特化業務型 AI** も使い分け、Deep Research・ファイル解析・画像生成・音声入力まで駆使する段階。

**具体的なタスク（汎用 AI 側）**
- **ChatGPT Deep Research / Perplexity** で競合 5 社の事業戦略を 30 ページレポート化
- **Claude** に Excel（売上データ 3 年分）を投げて、ピボット分析＋グラフ入り PPT 草案を作成
- **Gemini Advanced** で自分の Google ドライブ内を横断検索
- **Grok** で X（旧 Twitter）のリアルタイム情報を取得
- 音声モードで移動中にブレインストーミング

**具体的なタスク（特化業務型 AI 側）**
- **Notta / tl;dv** で商談を自動文字起こし→要約
- **Manus** で「業界レポート→スライド→PDF」を丸投げ
- **NotebookLM** に論文 10 本を投入しポッドキャスト化
- **Gamma** でプレゼン資料を自動生成
- **Nano Banana / Gemini Image** で提案書の表紙画像を生成
- **Firecrawl** で大量 PDF・Web ページから情報抽出

**使い分け早見表**
| 用途 | 第一候補 |
|:---|:---|
| 深掘り調査 | ChatGPT Deep Research / Perplexity |
| 長文・ファイル生成 | Claude |
| X の最新情報 | Grok |
| 議事録 | Notta / tl;dv |
| 資料作成 | Manus / Gamma |
| 音声入力 | Typeless |
| 自分のデータ横断検索 | Gemini Advanced / NotebookLM |

**卒業条件**：「この用途ならこの AI」の使い分けを**秒で**判断できる。

---

### Lv.III ボットビルダー — 「自分の業務を固定化する者」

**定義**：**GPTs / Gems / Claude Projects** などで自作 AI を作り、**自分の業務プロセスを再現・自動化・固定化**する段階。その過程で出来たボットを周囲に配布することで、チームの生産性まで底上げする。

**コンセプト**：毎回書いているプロンプト、何度も答えている質問、同じフォーマットの作業——これらを **ボットに閉じ込めて固定化** する。

**具体的なタスク**
- 自分の**週報テンプレート**を学習させた「週報生成 Bot」
- 自分の**提案書パターン 5 種**を読ませた「提案書ドラフト Bot」
- 過去の**議事録 100 本＋フォーマット**を学んだ「議事録整形 Gem」
- 自分の **X 投稿の書き方**を学習した「自分クローン GPT」
- 後輩からよく聞かれる質問を集めた「人事 FAQ ボット」を社内配布

**Before / After**
- Before：同じプロンプトを毎回書く／同じ質問に何度も答える／属人化
- After：Bot が自分の代わりに同じ品質で動く／休暇中もチームが詰まらない／**自分のやり方が資産として残る**

**ポイント**：「他人に配布する」より**「自分の仕事を閉じ込める」**が先。配布はその副産物。

**卒業条件**：**自分の定型業務の 80% 以上**が Bot 経由で回っている。

---

### Lv.IV アーティファクター — 「公開できる成果物を作る者」

> ★ **非エンジニアが最初に目指すべき到達ライン**

**定義**：Claude **Artifacts** / ChatGPT Canvas / Lovable / v0 などで、**URL 1 本で共有できるミニアプリ**を作れる段階。

**具体的なタスク**
- Claude Artifacts で「**自社サービスの料金シミュレーター**」を作成、営業資料に貼る
- 「業界向け AI 活用度診断」を Artifacts で作り SNS で配布しリード獲得
- ChatGPT Canvas で**インタラクティブな決算レポート**を作成、経営会議で使用
- Lovable で**社内イベント申込 LP** を 30 分で公開
- v0 で **BtoB 商材の比較表コンポーネント**を自社サイトに埋め込み

**成果物**：料金／ROI シミュレーター／診断クイズアプリ（SNS シェア付き）／社内ミニダッシュボード／申込 LP／インタラクティブ提案書

**Before / After**
- Before：PDF や PPT を添付して送る／「こんなイメージです」で終わる
- After：URL を送るだけで相手が**触って体験**できる／その場で数字を変えて試せる

**エンジニア／非エンジニアの差**：**ほぼゼロ**。ここが非エンジニアにとっての "作る側" の壁。

**卒業条件**：第三者に **URL を渡して実際に業務で使ってもらえる**アプリを 1 本作る。

---

### Lv.V バイブコーダー — 「自然言語でアプリを建てる者」

**定義**：Cursor / Lovable / Bolt / Claude Code を使い、自然言語主導で**社内ツール・簡単な社外向けツール**を作って運用する段階。

**具体的なタスク（社内向け）**
- 社内の**予実管理ツール**を Claude Code で構築、毎日 50 人が使用
- 営業部専用の**見積生成システム**を Lovable で 2 週間で公開
- 既存 Excel フローを **Web アプリ化**、権限管理・履歴付きにアップデート

**具体的なタスク（社外向け）**
- 業界特化の**ニッチ SaaS ベータ**を週末に公開、初月から課金ユーザー獲得
- **Stripe 決済**を組み込み、本当にクレカで売上が立つ状態を作る
- クライアントごとの**カスタムダッシュボード**を受託開発

**Before / After**
- Before：社内ツールを Excel ＋手作業で運用／外注に 3 ヶ月 100 万円
- After：週末に自分で作り、翌週から運用開始／初月から課金ユーザー獲得

**エンジニア／非エンジニアの差**
- エンジニア：週末に MVP を複数ローンチ、売れ筋を見極めて投資
- 非エンジニア：自分の業界の "見えてる課題" を SaaS 化、ドメイン知識で差別化

**卒業条件**：**有料ユーザー**または**社内本番運用**を持つアプリを運用している。

---

### Lv.VI オートメーター — 「業務を寝かせる者」

> ★ **非エンジニアが到達できる現実的な最上位ライン**

**定義**：自分の業務を**分解**し、Dify / Dify Auto / n8n / Make / Zapier / Claude Code / MCP で**ツール群を繋いで 24 時間自動稼働**させる段階。Lv.V で作ったツールや Lv.III の Bot も、ここで一本の自動ラインに統合される。

**コンセプト**：業務を「工程」に分解し、**どの工程をどの AI／ツールが担うか**を設計して、自分は手を動かさずに結果だけ受け取る。

**具体的なタスク**
- **メール受信 → AI 分類 → 重要度タグ付け → Slack 通知** のフローを n8n で構築
- **商談議事録 → 自動要約 → Salesforce 登録 → お礼メール下書き** を Zapier ＋ Claude で無人化
- **毎朝 7 時にニュース・Gmail・カレンダーを読み込み → Slack 通知＋音声ブリーフィング**
- **Dify Auto** で「こういう業務を自動化して」と自然言語指示するだけでワークフロー生成
- **MCP サーバー** を使い、Claude Desktop から Notion/Slack/Gmail を直接操作
- 顧客企業のワークフロー構築を請け負う（**AI コンサル**として月 30〜100 万円）

**成果物**：稼働中の n8n／Dify ワークフロー／毎朝届く Slack ブリーフィング／24 時間応答 RAG Bot／**月 40〜80 時間削減**を可視化したレポート

**Before / After**
- Before：毎朝メール確認 30 分／議事録整理 1 時間／毎週の定型作業数時間
- After：寝ている間に終わっている／通知を見るだけ／**月 40〜80 時間の削減が数値で見える**／自分が休んでも業務が止まらない

**卒業条件**：**自分が寝ている・休んでいる間も動いている**ワークフローを**複数本**運用。

---

### Lv.VII エージェントマスター — 「事業を AI に任せる者」

**定義**：自分の業務・事業を分解し、**Claude Code / Codex / Managed Agents / MCP** を駆使して、AI エージェント群に**事業単位のタスク**を丸ごと委譲する段階。

**具体的なタスク**
- 就寝前に Claude Code に「市場リサーチ→要件定義→モック→実装→コードレビュー」を一括指示、翌朝成果物をレビュー
- **複数のサブエージェント**を並列起動（リサーチ担当・実装担当・テスト担当に分割）
- **独自 MCP サーバー**を開発して公開（社内ツール／顧客システムを AI から操作可能に）
- **Skills / Hooks** を自作し、Claude Code 環境をチューニング
- **Claude Managed Agents / A2A** で本番エージェントを運用（認証・サンドボックス・権限・ログ・コストまで設計）
- AI を中核にした**新規事業**を立ち上げ、MRR を立てる

**成果物**：公開中の独自 MCP サーバー（GitHub スター付き）／自作 Skills・Subagent 定義（OSS 配布）／AI が前提の新規事業（MRR あり）／24 時間稼働エージェント基盤／**1 人× Claude Code で従来比 5〜10 倍の事業立ち上げ速度**

**Before / After**
- Before（Lv.VI）：自分が分解した業務を自動化する（指示書を書く側）
- After（Lv.VII）：**業務の分解と実行設計すらも AI に任せる**／事業立ち上げに 1 年＋数千万円 → 週末プロト→1 ヶ月ベータ→3 ヶ月収益化、しかも 1 人で複数並行

**卒業条件**：**人間 1 人では物理的に回らない量**の業務を AI 経由で回している／AI が前提の事業で MRR が立っている。

### Lv.VIII AI アルケミスト — 「AI を現実の価値と収益に錬成する者」

**定義**：Lv.VII までに手に入れた「ツール化・自動化・エージェント運用」の力を束ね、**AI を中核にしたプロダクトを市場に出荷し、現実に収益を生み続ける**段階。技術者であると同時に**クリエイター／事業家**。

**この段階で成り立つ 3 つの型**
1. **社外向け SaaS 型**：AI-native の有料プロダクトを運営し、有料ユーザー数百〜数千人、月次売上が立っている
2. **基幹システム型**：自社／顧客企業の**基幹業務**を AI が担い、毎日数十〜数百人が業務として稼働に依存している
3. **プロダクト量産型**：複数の AI サービスをポートフォリオ運営し、累計売上・広告収益・ライセンス収益を積み上げている

**具体的なタスク**
- 独自 AI-native SaaS をローンチし、**Stripe 等で決済、有料ユーザーを獲得**
- **AWS / GCP / Azure** 等のハイパースケーラー上で本番環境を設計・運用（IAM・VPC・コスト最適化まで掌握）
- **Bedrock / Vertex AI / Azure OpenAI** や各社 Fine-tuning 基盤で独自モデル推論を本番稼働
- 自社の**基幹業務システム**を AI 駆動で構築、社員／顧客が日常利用
- AI Lab（小規模・人間＋ AI ハイブリッド）で**複数プロダクトを並行ローンチ**
- プロダクトの**ブランディング・UX・カスタマーサポート**までを設計

**成果物**：MRR が立っている AI-native SaaS（月商 10 万〜数千万円）／社内基幹として稼働する AI システム（クラウド上で SLA 担保）／複数プロダクトのポートフォリオ／「AI だから」ではなく「価値があるから」選ばれる完成度のプロダクト

**求められるもの**
- 「完成させて世に出す」意志（Lv.VII までの技術力を、プロダクトに翻訳する）
- ユーザーに対する責任（本番稼働、カスタマーサポート、継続改善）
- クリエイターとしての**審美眼**（AI だけでは埋まらない、ブランド・UX・物語）

**卒業条件**：自社の AI プロダクト／サービスが、**現実に収益を生み続けている**。

### Lv.IX シンギュラリアン — 「AI に会社そのものを動かさせる者」

> ★ **AGI 時代の到達地点。2026 年時点ではまだ誰も完成形に達していない仮説レベル。**

> Lv.VIII までは「人間＋ AI で事業を作る」領域。Lv.IX は **「AI が事業そのものを動かす」** という質的跳躍がある。

**定義**：AGI 級の AI／フルオートエージェント群が、**会社そのもの**を運営し、**意思決定・営業・開発・サポート・経理・採用・マーケティングまでを自律実行**、**収益までもが自動化**されている状態。

**この段階で AI が担うもの**
- **経営判断**：市場データ・競合・内部 KPI を読んで戦略を立案、意思決定
- **営業**：リード生成→商談→契約→請求までフルオート
- **プロダクト開発**：課題発見→要件定義→実装→リリース→改善まで自走
- **カスタマーサポート**：24 時間多言語対応、クレーム検知・エスカレーション
- **経理・法務**：仕訳・請求・契約レビュー・税務まで AI が担当
- **マーケティング**：トレンド検知→コンテンツ生成→配信→ CV 分析→次施策まで無人

**人間（創業者）の役割**
- ビジョン設定と**価値観の注入**
- AI 同士の判断が割れた時の**最終意思決定者**
- 法的・倫理的な**最終責任の所在**
- **社会との接続**（メディア・規制当局・業界）

**成果物**：AI が単独運営する事業（人間の関与が週数時間以下で、MRR が成長）／独自推論基盤／独自ファインチューニングモデル／**"AI Cofounder"** として立ち上がった会社

**求められるもの**
- 高貴なる者の義務（noblesse oblige）
- AI が社会に与える影響への**責任と説明**
- 自社だけでなく **AI 業界全体／人類全体**の利益を優先する視座

## 設計思想

### なぜ「格付け」か？

- 「ChatGPT を使っています」だけでは、実態のレベル差 **10 倍以上** を捉えられない
- 採用の現場・自己評価・社内研修設計で、共通の物差しが必要
- 単なる理論ではなく、**2026 年時点の最新 AI ツール群**（Claude Code / Artifacts / Manus / Lovable / Dify Auto / OpenClaw 等）を実戦的に反映

### なぜ「世界共通」か？

AI 熟達度は国境を越える普遍的な指標です。
このリポジトリでは、国・業界・職種を越えた **共通言語** としてのランクシステムを目指します。

### 4 つの原則

1. **実行力 ＞ 知識量** — 「知っている」ではなく「回している」で測る
2. **公開／配布 ＞ 自分用** — 自分専用で終わるものは 1 段下がる（ただし Lv.III は自分用が起点）
3. **継続稼働 ＞ 単発** — 24 時間回っている仕組みを持つ者が上位
4. **他者貢献 ＞ 自己効率化** — 他者・業界・人類の生産性まで上げられる者が最上位

### エンジニア／非エンジニア 到達ライン比較

| Lv. | 非エンジニアの到達ライン | エンジニアの到達ライン |
|:---:|:---|:---|
| I〜III | 同じ（業務の固定化まで） | 同じ |
| IV アーティファクター | ★ **最初のゴール**：Artifacts で顧客に価値提供 | 通過点 |
| V バイブコーダー | 社内ツール／簡易 SaaS を自分で作る | 現場のスタンダード |
| VI オートメーター | ★ **現実的な最終到達ライン**：業務を 24h 無人化 | 通過点 |
| VII エージェントマスター | プロダクトリードとして到達（AI 事業を設計） | ★ **本丸**：Claude Code / MCP を制する |
| VIII AI アルケミスト | AI プロダクトで収益化（クリエイター／事業家として） | ★ **商業的本丸**：AI SaaS・基幹システムをシップ |
| IX シンギュラリアン | AGI 時代の起業家として到達 | 最終地点：AI インフラの提供者 |

> 非エンジニアの現実的な最終形態 = **Lv.IV〜VI**
> エンジニアの到達目標 = **Lv.VII〜VIII**
> 人類が目指す最終形態 = **Lv.IX**

## 自己診断チェックリスト

以下のうち、**「実際にやったことがある」** ものをカウント：

- [ ] ChatGPT/Gemini/Copilot の**無料版**を業務で使っている … **Lv.I**
- [ ] 汎用 AI の**有料プラン**＋**Notta / Manus / NotebookLM 等の特化型 AI** を使い分けている … **Lv.II**
- [ ] **GPTs / Gems / Claude Projects** で自分の業務を固定化し、自分が一番のユーザーになっている … **Lv.III**
- [ ] **Claude Artifacts / Lovable / v0** で**公開 URL を持つアプリ**を作ったことがある … **Lv.IV**
- [ ] **Cursor / Lovable / Claude Code** で社内本番／社外デプロイ済みアプリを持っている … **Lv.V**
- [ ] **Dify / Dify Auto / n8n / Zapier / MCP** で 24 時間稼働のワークフローを複数運用している … **Lv.VI**
- [ ] **Claude Code / Managed Agents / MCP** で事業レベルのタスクを AI に委譲している … **Lv.VII**
- [ ] AI で作ったサービスが**現実に収益**を生み、市場／社内基幹として稼働している … **Lv.VIII**
- [ ] AI が**会社そのものを運営**し、**収益まで自動化**されている … **Lv.IX**（AGI 到達後の未来像）

最高チェック Lv. が、あなたの現時点のランクです。診断は [ai-rank.org](https://ai-rank.org) で 2 分で受けられます。

## 📊 診断の仕組み

- 10 問のマルチステップクイズ（各問 4 択）
- 回答クリックで自動的に次の質問へ（350ms ディレイ）
- Q1〜Q9：各問が Lv.I〜Lv.IX の判定
- Q10：業界貢献度のボーナスバッジ
- スコア 2 以上（C または D 回答）= その Lv. 到達と判定
- 最終ランク = 到達した最高 Lv.

診断結果は、名前入り・シリアル番号入りの **認定証** として発行されます。
認定証はインタラクティブカードとしてインライン表示されると同時に、ランク別にユニークな URL（`/c?rank=N&name=X`）でシェア可能な OG 画像も生成されます。

## 🔐 プライバシー

- 診断結果はブラウザの `localStorage` に保存（サーバーには送信されません）
- SNS シェア／PNG 保存時のみ、**氏名・メール・会社名** の登録をお願いしています
- 登録情報は証明書発行・シェア機能、法人向け診断ツール（有料・準備中）の先行案内のみに使用
- トラッキング Cookie・サードパーティ解析はデフォルトで未導入
- 詳細は [docs/DATA_STORAGE.md](./docs/DATA_STORAGE.md) 参照
