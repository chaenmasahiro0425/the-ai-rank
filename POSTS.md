## 🧵 パターンB【スレッド形式・本命】（7ツイート）

### 1/7 — Hook

```
あなたのAIレベル、何段だと思いますか？

「個人のAI活用スキル」を測る世界共通の物差しが
この世に存在しなかったので、作りました。

─── THE AI RANK ───
9段階のAI格付けランク · by CHAEN
https://ai-rank.org

オープンソースで、完全無料で公開します。
🧵続きます👇
```

### 2/7 — 背景

```
【なぜ作ったか】

・「AI使える人」の定義が会社ごとにバラバラ
・採用で「AI得意です」と言われても実力が分からない
・社内研修の到達点を決められない
・"AI人材"が曖昧なまま給与設計される

共通の物差しが無いから、現場が苦しんでる。
じゃあ、作ろう。
```

### 3/7 — Lv.I〜III（基礎層）

```
【Lv.I 〜 III — 基礎層】

Ⅰ ビギナー
　無料版を触り始めた段階

Ⅱ パワーユーザー
　有料＋特化AIを用途別に使い分ける

Ⅲ ボットビルダー
　GPTs/Gemsで自分の業務を固定化

ここまでは「使う側」。
多くの社会人が Lv.II 前後で止まっている。
```

### 4/7 — Lv.IV〜V（非エンジニアの壁）

```
【Lv.IV 〜 V — 作る側への突破】

Ⅳ アーティファクター
　Artifactsで"自分ベース"のミニアプリを作り、
　他人に使われ始める（非エンジニアの第一ゴール）

Ⅴ バイブコーダー
　Cursor/Lovable/Claude Codeで本番運用される
　ツールを自然言語で作る

ここから「作る側」に入る。
```

### 5/7 — Lv.VI〜IX（上位）

```
【Lv.VI 〜 IX — 上位層】

Ⅵ オートメーター
　24h自動稼働のワークフローを回す

Ⅶ エージェントマスター
　事業単位のタスクをAIエージェント群に委譲

Ⅷ AIアルケミスト
　AIで創ったプロダクトが収益を生み続ける

Ⅸ シンギュラリアン
　AIが"会社そのもの"を運営する（未達・仮説）
```
